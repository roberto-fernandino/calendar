This is a [Next.js](https://nextjs.org/) calendar project.

## Getting Started

First, install the dependencies:

```bash
npm install
#or
npm i
```

Then, start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the calendar.
